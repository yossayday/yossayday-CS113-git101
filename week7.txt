Week 7: Pointers and Dynamic Memory Allocation

Pointer คือ ตัวแปรที่ทำการเก็บค่าที่อยู่ของข้อมูลตัวแปร การใช้ Pointer นั้นจะทำให้เราสามารถเข้าถึงข้อมูลได้โดยตรงโดยใช้ที่อยู่ของหน่วยความจำ
แทนที่จะใช้ชื่อของตัวแปรที่เราได้ใช้

Address of operator (&)
การใส่สัญลักษณ์ ampersand (&) จะทำให้เราได้รับค่าที่อยู่ของตัวแปรในหน่วยความจำที่อยู่นี้สามารถได้รับมาได้ในตอนที่โปรอกรมรันเท่านั้น

Dereference operator (*)
ในการเข้าถึงข้อมูลของตัวแปร Pointer เราจะใช้เครื่องหมาย star (*) หน้าตัวแปร Pointer เพื่อเข้าถึงข้อมูลที่อยู่ใน Pointer 

คำสั่ง scanf() เป็นการรับ input เข้ามาเพื่อ process
ใช้ได้โดยรูปแบบ
int a;
scanf("%d",&a);
