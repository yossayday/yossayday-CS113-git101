week 8: Linked List

linked list นั้นมีความคล้ายคลึงกับ array แต่จะมีความแตกต่างงกันอยู่เล็กน้อย
Array นั้นในแต่ละกล่องจะใส่ค่าที่เราใสาลงไป ส่วน linked list นั้นจะเก็บ address ของช่องที่ติดกันด้วยมีทั้งรูปแบบทางเดียวและไปกลับได้

linked list มี 2 แบบ
- linked list ทางเดียว
เราไม่สามารถเรียกกลับข้อมูลก่อนหน้านี้ได้ใน สinked list ทางเดียวเนื่องจากเรามี pointer ที่ชี้ไปยังกล่องถัดไปเท่านั้นซึ่งถ้าเราต้องการข้อมูลก่อนหน้านั้นจำเป็นต้องไล่จากหัวใหม่เสมอ
- linked list สองทาง
จะสามารถเรียกข้อมูลก่อนหน้านี้มาได้เนื่องจากในกล่องมีทั้ง pointer ที่ชี้ไปยังกล่องถัดไป และ กล่องก่อนหน้า

linked list จะมีกล่องเก็บข้อมูล ซึ่งกล่องเก็บข้อมูลจะเรียกว่า node ซึ่งเราจะใช้ struct สร้างขึ้นมาม
มีโครงสร้างดังนี้

struct node {
     int data;
     struct node *next;  
};
